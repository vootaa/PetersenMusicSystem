这是一个基于中国传统五行阴阳理论和黄金比例的音阶生成系统库。

以下是其主要作用和特点：

核心功能
Petersen可变比例音阶生成器，实现了一套独特的音律系统，
结合了：
    五行理论（金、木、水、火、土）
    阴阳极性（阴、中、阳）
    可变比例φ（如黄金比例1.618、八度2.0等）
    几何角度分布

主要特性
1. 灵活的参数化系统
    16个φ值预设：从微分音到大音程（黄金比例、八度、完全五度等）
    21个δθ值预设：覆盖各种特殊几何结构（15等分、五角星等）
    每个360°音区内生成15个音位（5个五行 × 3个极性）

2. 完整的音阶生成算法

3. 多种导出格式
    CSV格式：详细数据表格
    Scala (.scl)格式：专业音律软件格式
    MIDI调音(.tun)格式：MIDI设备调音表

4. 强大的分析功能
    音程关系分析
    频率重叠检测
    统计信息生成
    几何美学描述

使用场景
    音乐创作
        冥想音乐：黄金比例系统
        现代音乐：八度关系兼容12平均律
        实验音乐：各种微分音系统
        治疗音乐：完全五度系统

    音乐研究
        微分音律研究
        东方音律系统
        数学与音乐关系研究

核心API示例
# 创建不同类型的音阶
    scale_golden = PetersenScale(phi=1.618)    # 黄金比例
    scale_octave = PetersenScale(phi=2.0)      # 八度关系  
    scale_fifth = PetersenScale(phi=1.5)       # 完全五度

# 生成和分析
    entries = scale.generate()                     # 生成音阶

创新之处
这个库将古代东方哲学（五行阴阳）与现代数学（黄金比例、几何学）结合，创造了一个既有传统文化底蕴又具备现代科学精度的音律生成系统，为音乐创作和研究提供了全新的工具。
