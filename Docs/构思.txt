定制 Petersen / Chainweb 钢琴设计总结

1. 键盘结构
	•	白键：Petersen 数学公式映射的基础音阶（核心音阶）
	•	黑键：根据核心比率（3/2、5/4、6/5 等）构建和弦音，如果不在白键上就新增为黑键 → 形成动态扩展音阶
	•	特点：白键稳定，黑键一次性生成 → 静态、全局固定

⸻

2. 和弦设计
	•	根音（白键） + 核心比率 → 构成和弦
	•	如果和弦音不在音阶表 → 直接增加黑键
	•	保持频率比接近谐波 → 听感更和谐
	•	和弦可以跨层（内圈/中圈/外圈节点）增强空间感

⸻

3. 旋律与多声部
	•	低音（Bass）：慢节奏，提供节拍锚点
	•	和弦（Chord）：中等节奏，填充和声
	•	旋律（Melody）：快速移动，可在低音周期内多次起伏，形成“飞舞”效果
	•	多声部可通过不同周期整合 → 合拍、丰富层次感

⸻

4. 立体空间约束（Chainweb / 三层圆圈）
	•	20 个节点，3 层同心圆：
	•	内圈：5 个节点（五行本位）
	•	中圈：5 个节点（过渡层）
	•	外圈：10 个节点（五行阴阳位）
	•	15 个方位对应 15 个音高
	•	双向连接 60 条，节点间可以随机游走
	•	同方位节点上下可连 → 形成临近音区关系
	•	旋律走向 = 空间约束 + 随机/模式游走

⸻

5. 旋律模式
	•	对称性：中心对称、轴对称、频率对称
	•	循环/变形 motif：重复旋律片段，增强秩序感
	•	渐进/渐退：旋律沿网络上升或下降
	•	概率偏好：游走时偏好和谐音程 → 提高音乐可听性

⸻

6. 节奏设计
	•	五方位制度 → 拍子以 5 为基准
	•	阴阳位 → 对应强弱拍：
	•	阳位 → 强拍（低音、和弦主音）
	•	阴位 → 弱拍（旋律装饰或弱音）
	•	周期层次：低音慢，和弦中，旋律快 → 多声部自动合拍

⸻

7. 可听性优化
	•	核心比率 + 微调频率 → 保持和谐
	•	模式约束 + 重复 motif → 增加熟悉感
	•	概率偏好游走 → 降低刺耳跳跃
	•	多声部分层 → 稳定节奏 + 丰富旋律 + 动态和声

⸻

8. 演奏逻辑总结
	1.	白键作为基础音阶，黑键作为动态扩展
	2.	低音推进 → 节拍锚点
	3.	和弦填充 → 多声部和声支撑
	4.	旋律飞舞 → 在立体网络中随机或规则游走
	5.	节奏周期 + 阴阳强弱 → 合拍且有律动
	6.	模式（对称、循环、渐进）+ 空间约束 → 保持音乐可听性

⸻

✅ 整体效果
	•	一台逻辑自洽、空间受约束的定制钢琴
	•	多声部、多节奏、多模式的组合
	•	即便是非十二平均律（Petersen 音阶），仍然大概率可听且有音乐性
