定制 Petersen / Chainweb 钢琴设计总结
====================================================================

1. 键盘结构
	•	白键：Petersen 数学公式映射的基础音阶（核心音阶）
	•	黑键：根据核心比率（3/2、5/4、6/5 等）构建和弦音，如果不在白键上就新增为黑键 → 形成动态扩展音阶
	•	特点：白键稳定，黑键一次性生成 → 静态、全局固定

⸻

2. 和弦设计
	•	根音（白键） + 核心比率 → 构成和弦
	•	如果和弦音不在音阶表 → 直接增加黑键
	•	保持频率比接近谐波 → 听感更和谐
	•	和弦可以跨层（内圈/中圈/外圈节点）增强空间感

⸻

3. 旋律与多声部
	•	低音（Bass）：慢节奏，提供节拍锚点
	•	和弦（Chord）：中等节奏，填充和声
	•	旋律（Melody）：快速移动，可在低音周期内多次起伏，形成“飞舞”效果
	•	多声部可通过不同周期整合 → 合拍、丰富层次感

⸻

4. 立体空间约束（Chainweb / 三层圆圈）
	•	20 个节点，3 层同心圆：
	•	内圈：5 个节点（五行本位）
	•	中圈：5 个节点（过渡层）
	•	外圈：10 个节点（五行阴阳位）
	•	15 个方位对应 15 个音高
	•	双向连接 60 条，节点间可以随机游走
	•	同方位节点上下可连 → 形成临近音区关系
	•	旋律走向 = 空间约束 + 随机/模式游走

⸻

5. 旋律模式
	•	对称性：中心对称、轴对称、频率对称
	•	循环/变形 motif：重复旋律片段，增强秩序感
	•	渐进/渐退：旋律沿网络上升或下降
	•	概率偏好：游走时偏好和谐音程 → 提高音乐可听性

⸻

6. 节奏设计
	•	五方位制度 → 拍子以 5 为基准
	•	阴阳位 → 对应强弱拍：
	•	阳位 → 强拍（低音、和弦主音）
	•	阴位 → 弱拍（旋律装饰或弱音）
	•	周期层次：低音慢，和弦中，旋律快 → 多声部自动合拍

⸻

7. 可听性优化
	•	核心比率 + 微调频率 → 保持和谐
	•	模式约束 + 重复 motif → 增加熟悉感
	•	概率偏好游走 → 降低刺耳跳跃
	•	多声部分层 → 稳定节奏 + 丰富旋律 + 动态和声

⸻

8. 演奏逻辑总结
	1.	白键作为基础音阶，黑键作为动态扩展
	2.	低音推进 → 节拍锚点
	3.	和弦填充 → 多声部和声支撑
	4.	旋律飞舞 → 在立体网络中随机或规则游走
	5.	节奏周期 + 阴阳强弱 → 合拍且有律动
	6.	模式（对称、循环、渐进）+ 空间约束 → 保持音乐可听性

⸻

✅ 整体效果
	•	一台逻辑自洽、空间受约束的定制钢琴
	•	多声部、多节奏、多模式的组合
	•	即便是非十二平均律（Petersen 音阶），仍然大概率可听且有音乐性


====================================================================
非立体结构，避免跨音区的突兀感
定制 Petersen 图钢琴音乐设计总结
====================================================================

1. 键盘与音阶
	•	白键：Petersen 图映射的基础音阶（核心音阶）
	•	黑键：动态生成的和弦音（基于核心比率 3/2、5/4、6/5 等），如果不在白键上则新增黑键
	•	特性：白键稳定，黑键一次性生成 → 静态、全局固定
	•	频率微调：为了提升音乐可听性，黑键或和弦音可微调至接近理想比率

⸻

2. 和弦构建
	•	根音（白键）+ 核心比率 → 和弦
	•	和弦音可以跨层（同音区内邻近节点）
	•	若和弦音不在音阶表 → 直接加入黑键
	•	保持频率比接近纯律 → 听感和谐

⸻

3. 旋律与多声部
	•	低音（Bass）：慢节奏，提供节拍锚点
	•	和弦（Chord）：中等节奏，填充和声
	•	旋律（Melody）：快速移动，在低音周期内多次起伏 → “旋律飞舞”
	•	多声部周期设计使各层合拍（低音慢，和弦中，旋律快）

⸻

4. 局部旋律约束
	•	同音区内部：使用 Petersen 图的 30 条无向边 / 60 条单向边
	•	节奏和模式可增加对称性、循环 motif、渐进/渐退等结构
	•	跨节点移动可根据概率偏好 → 偏向和谐音程（避免刺耳跳跃）

⸻

5. 跨音区连接
	•	音阶递升/递降 → 下一音区的起点按照五方位 + 阴阳顺序选择
	•	例如：土阳 → 金阴 → 水阳 …
	•	跨音区可通过：
	1.	过渡音 / 分解音平滑连接
	2.	弱拍触发 → 降低突兀感
	3.	概率偏好 → 遵循音阶顺序
	•	保证旋律连续性，避免直接跨八度的突兀感

⸻

6. 阴阳与节拍
	•	五方位制度 → 拍子以 5 为基准
	•	阴阳位 → 对应强弱拍：
	•	阳位 → 强拍（低音、和弦主音）
	•	阴位 → 弱拍（旋律装饰或弱音）
	•	多声部与阴阳位结合 → 节奏律动自然，旋律可多次起伏

⸻

7. 可听性优化
	•	核心比率 + 微调 → 保持和谐
	•	模式约束 + 循环 motif → 增强熟悉感
	•	概率偏好游走 → 降低突兀跳跃
	•	多声部分层 → 稳定节奏 + 丰富旋律 + 动态和声

⸻

8. 演奏逻辑概览
	1.	白键 → 基础音阶；黑键 → 动态和弦
	2.	低音慢推进 → 提供节拍锚点
	3.	和弦填充 → 多声部和声支撑
	4.	旋律在 Petersen 图内部游走 → 同音区动态约束
	5.	音阶递升/递降 → 跨音区平滑连接
	6.	阴阳位与五方位拍子 → 强弱节奏自然
	7.	模式约束（对称、循环、渐进） → 提高音乐可听性

⸻

✅ 整体效果
	•	多声部、多节奏、多模式的定制钢琴演奏
	•	局部受 Petersen 图约束 → 旋律有规律
	•	全局递升/递降 + 阴阳拍子 → 跨音区顺滑、节奏清晰
	•	即便在非十二平均律下，也能大概率听得悦耳、结构清晰

=========================================================
定制钢琴节奏总结
	1.	基本单位
	•	最小时间单位 = 1/15 拍 → 对应 15 个音符/小节
	2.	小节结构
	•	小节长度 = 5 拍 → 对应 5 个根音（五方位：金、木、水、火、土）
	•	每拍内部可走 3 个音符 → 根音 + 阴位音 + 装饰音/旋律音
	3.	强弱拍（阴阳位）
	•	阳位 → 强拍 → 根音或低音节点
	•	阴位 → 弱拍 → 旋律装饰音或过渡音
	4.	声部层次
	•	低音：慢节奏，每拍走根音
	•	和弦：中节奏，填充弱拍或强拍辅助音
	•	旋律：快节奏，每拍 3 个音符 → 飞舞、丰富
	5.	循环与律动
	•	小节内旋律多次起伏 → 保持动感
	•	根音顺序对应五方位 → 稳定循环
	•	可跨音区递升/递降 → 顺滑连接下一小节 (只发生在金阴和土阳之间，或者说金和土之间)

补充：
为了低音、和弦、旋律多声部协调，最小单位可以设为 1/30 拍，在每拍内部精确分配各声部位置，使低音稳固、和弦填充自然、旋律自由飞舞。

⸻

✅ 核心思路：
五方位根音提供节拍锚点 + 阴阳位产生强弱拍 + 每拍多音符形成旋律飞舞，多声部自然合拍且富有律动感。
